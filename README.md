Social Media Platform
A fully functional front-end Social Media Platform built using HTML, CSS, and JavaScript. The project demonstrates core social networking features including user profiles, follow/unfollow functionality, post creation, likes, comments, activity tracking, and data persistence using Local Storage.

How to Run
Download or clone the repository.
Open the project folder.
Run the index.html file in any modern web browser.
Start interacting with the social media platform.
Database Design
The application uses a JavaScript object-based database structure:

Users
Posts
Likes
Comments
Followers
Following
All data is stored locally in the browser using Local Storage.
